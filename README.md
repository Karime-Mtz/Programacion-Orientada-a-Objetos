## Programacion-Orientada-a-Objetos

# Problemática
Mejorar la organización y administración de una biblioteca digital, tomando en cuenta tres aspectos:
Libros: su información básica
Usuarios: sus datos y las acciones que pueden realizar dentro de la biblioteca
Valoración: las reseñas y comentarios que usuarios han escrito respecto a libros

# Función del Programa
La función de este programa es demostrar que se pueden modelas sistemas dela vida real a través de la Programación Orientada a Objetos.

# Modo de Uso
El usuario puede usar este programa, sobre todo para obtener información acerca de un libro y sus reseñas.
