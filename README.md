# Programacion-Orientada-a-Objetos

## Problemática
Mejorar la organización y administración de una biblioteca digital, tomando en cuenta tres aspectos:

Libros: su información básica
Usuarios: sus datos y las acciones que pueden realizar dentro de la biblioteca
Valoración: las reseñas y comentarios que usuarios han escrito respecto a libros

## Función del Programa
La función de este programa es demostrar que se pueden modelas sistemas de la vida real a través de la Programación Orientada a Objetos.
Se crean 3 clases relacionadas a una Biblioteca con funciones que representan las acciones que se pueden realizar en la vida cotidiana. 

## ¿Cómo se debe usar?
El usuario puede usar este programa, sobre todo para obtener información acerca de un libro y sus reseñas. Se crean objetos de cada clase, 
ya sean libros, usuarios o reseñas, y cada uno puede realizar ciertas acciones. Por ejemplo, se puede ver la información de un libro, se puede
agregar una reseña y se puede establecer el género favorito de lectura de un usuario.
