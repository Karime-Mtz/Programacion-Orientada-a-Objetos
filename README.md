# Programacion-Orientada-a-Objetos

## Problemática
Mejorar la organización y administración de una biblioteca digital, tomando en cuenta tres aspectos:

Libros: su información básica
Usuarios: sus datos y las acciones que pueden realizar dentro de la biblioteca
Valoración: las reseñas y comentarios que usuarios han escrito respecto a libros

## Función del Programa
Este programa funciona como una biblioteca digital, en la cual el usuario puede realizar acciones como buscar libros, agregarlos a la biblioteca, crear reseñas, ver su información, al igual que pedir libros prestados y devolverlos. Consta 
de un menu, a través del cual el usuario puede realizar las acciones que él desee hasta que decida salir del programa.

## Instrucciones
Compilar con: `g++ main.cpp -o main`

Correr en windows: `.\/main.exe´
